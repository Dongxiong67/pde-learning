# 学习心得

记录 PDE 学习过程中的思考、感悟与思想深线。来源于课堂纪要、教材深度挖掘与个人反思。

---

## 一、老师的"PDE 观念"——适定性是组织原则

老师在课堂上反复强调：偏微分方程主要研究解的"存在性、唯一性、稳定性"。这不是随口说的，而是整章位势方程的组织原则。

| 适定性问题 | 工具 | 解决的问题 |
|---|---|---|
| 存在性 | 基本解、Green 函数、Poisson 核 | 写出解的表示公式 |
| 唯一性 | 最大值原理、比较原理、能量法 | 两个解作差证明为零 |
| 稳定性 | 最大模估计、连续依赖 | 边界/源项扰动控制解 |
| 正则性 | 平均值性质、导数估计、能量 | 调和函数光滑性 |

老师讲法的深层逻辑：

1. 先证明调和函数很"刚性"——平均值性质、Harnack、最大值原理、导数估计。
2. 再用基本解/Green 函数把非齐次问题"表示出来"——解等于源项积分加边界积分。
3. 再用极值原理回答唯一性和稳定性——解不会在内部凭空变大。
4. 最后用能量估计进入更一般 PDE 的语言——逐点控制不够时，用积分能量控制。

**学习体会**：理解 PDE 不是背公式，而是建立"问题—工具"的对应关系。看到一个新方程，先问"它有哪种适定性问题？哪种工具最自然？"

## 二、第一条深线：调和函数不是"会算"，而是"被平均值锁死"

### 老师从平均值性质切入

纪要开头反复强调：

- 调和函数在球心的值等于球内平均值；
- 也等于球面平均值；
- 这是后面 Harnack、不等式、极值原理、导数估计的根。

教材/讲义的逻辑是：
$$\Delta u = 0 \Rightarrow u(x) = \fint_{\partial B(x,r)} u \, dS \Rightarrow u(x) = \fint_{B(x,r)} u \, dy.$$

### 一个技巧贯穿全章

老师讲证明时抓住一个技巧：**把含 $R$ 的变动区域通过变量替换变成单位球/单位球面，让 $R$ 进入被积函数，然后对 $R$ 求导**。

这条技巧非常重要——后面能量估计也重复用它（半径能量单调性证明）。

### 从平均值到极值原理

如果 $u$ 在内部点 $x_0$ 取得最大值，那么平均值公式说：
$$u(x_0) = \fint_{B(x_0,r)} u(y) \, dy.$$

但积分平均不可能等于最大值，除非球内几乎处处都等于最大值；由连续性推出局部恒等，再由连通性推出全域恒等。

**学习体会**：区域"连通"不是装饰条件。它用于从局部常数推进到整个区域常数。在写证明时，要主动写"由 $\Omega$ 连通，$O$ 既开又闭且非空，故 $O = \Omega$"。

### Harnack 的位置

Harnack 不等式不是期中最可能的大证明，但它说明一个思想：**非负调和函数在有界连通区域中不能一处很大、一处很小**。它和最大值原理属于同一类"调和函数刚性"结果。

## 三、第二条深线：导数估计是平均值性质的高级版本

### 老师为什么类比复变函数

老师说：解析函数有 Cauchy 导数公式，导数可由函数本身控制；调和函数作为解析函数实部/虚部的推广，也应有类似估计。

调和函数导数估计的精神：
$$|D^\alpha u(x)| \le \frac{C_{n,k}}{R^{n+k}} \int_{B(x,R)} |u(y)| \, dy, \qquad |\alpha| = k.$$

这说明：

1. 调和函数自动光滑；
2. 内部导数由稍大球上的 $L^1$ 信息控制；
3. 越靠近边界，估计越差（$R$ 不能超过到边界的距离）。

### 证明结构：老师讲得很细

证明分三层：

1. **$k = 0$**：就是平均值公式取绝对值。
2. **$k = 1$**：$\partial_i u$ 仍是调和函数，对 $\partial_i u$ 用平均值公式，再把 $\partial_i u$ 看成某个向量场的散度，用散度定理把体积分变成球面积分，最后对半径 $s \in (0, R)$ 积分。
3. **一般 $k$**：数学归纳法。把 $D^\alpha u$ 写成 $\partial_i D^\beta u$，$|\beta| = k - 1$。先在小球 $B(y, (1-t)R)$ 上用归纳假设，再放大到 $B(x, R)$，用嵌套球控制区域。

### 嵌套球技巧

$$y \in B(x, tR), \quad z \in B(y, (1-t)R) \Rightarrow z \in B(x, R).$$

证明：$|z - x| \le |z - y| + |y - x| < (1-t)R + tR = R$。

**学习体会**：这种"几何包含 + 积分放大"是教材证明里的关键细节。考试如果问"为什么嵌套球成立"，要会写这个三角形不等式。

## 四、第三条深线：从径向解到基本解，再到 Green 函数

### 老师如何引入基本解

纪要中"镜像对称与拉格拉斯方程"对应教材基本解部分。老师先问：如果解只依赖 $r = |x|$，即 $u(x) = v(r)$，那么 $\Delta u = 0$ 变成什么 ODE？

计算得到：
$$\Delta u = v''(r) + \frac{n-1}{r} v'(r).$$

所以径向调和函数满足：
$$v'' + \frac{n-1}{r} v' = 0.$$

令 $w = v'$，得 $w = C r^{1-n}$。积分：

- $n = 2$：$v(r) = C \log r + C_2$；
- $n \ge 3$：$v(r) = C r^{2-n} + C_2$。

老师特别指出：**这些非平凡径向解在 $r = 0$ 有奇性，所以它们不是全空间普通调和函数，而是在 $\mathbb{R}^n \setminus \{0\}$ 上调和**。这正是基本解的来源。

### 基本解的本质

基本解 $\Gamma$ 满足 $-\Delta \Gamma = \delta_0$。含义是：除了原点外它调和；在原点集中一个单位源。

**学习体会**：理解基本解不是普通逐点等式，而是分布意义下的等式。考试如果问"$-\Delta \Gamma = \delta_0$ 是什么意思"，要主动写"分布意义下"，并解释挖掉小球 $B(x, \varepsilon)$ 处理奇点再令 $\varepsilon \to 0$。

## 五、第四条深线：Green 函数是"基本解 + 边界修正"

### 老师提醒：Green 公式和 Green 函数不是一回事

纪要里老师专门提醒：**Green 公式是积分恒等式，Green 函数是用来写解的核函数**。

Green 函数的核心定义：
$$\begin{cases} -\Delta_x G(x, y) = \delta_y & \text{in } \Omega \\ G(x, y) = 0 & \text{on } \partial\Omega \end{cases}$$

它可以理解为 $G(x, y) = \Gamma(x - y) - H(x, y)$，其中 $H$ 用来把边界值修正为 0。

### 表示公式的讲义逻辑

固定 $x$，取 $v(y) = G(x, y)$，用 Green 第二公式：
$$\int_\Omega (u \Delta G - G \Delta u) = \int_{\partial\Omega}(u \partial_n G - G \partial_n u).$$

由于 $G = 0$ on $\partial\Omega$，边界项中 $G \partial_n u$ 消失。再用 $-\Delta G = \delta$ 和 $-\Delta u = f$ 得：
$$u(x) = \int_{\partial\Omega} g \partial_{n_y} G \, dS_y + \int_\Omega f G \, dy.$$

**学习体会**：考试如果问"为什么固定 $x$ 把 $y$ 当变量"，答案是 $G(x, y)$ 对 $y$ 满足 $-\Delta_y G(x, y) = \delta_x(y)$，Green 第二公式是在积分变量 $y$ 上使用的。

## 六、第五条深线：Poisson 核是球上 Green 函数的显式边界核

老师在纪要中讲了球上的 Poisson 公式：
$$u(x) = \int_{\partial B_R} P_R(x, y) g(y) \, dS_y.$$

Poisson 核的结构：
$$P_R(x, y) = \frac{R^2 - |x|^2}{n \alpha_n R |x - y|^n}, \qquad y \in \partial B_R.$$

老师强调的不是死背常数，而是：

1. 分子 $R^2 - |x|^2$：靠近边界时反映边界极限；
2. 分母 $|x - y|^n$：来自基本解法向导数；
3. 它是边界上的积分核。

**学习体会**：Poisson 核可以看成 $\partial_{n_y} G(x, y)$ 的显式形式。考试如果问"球上的 Green 函数对应什么核"，答 Poisson 核。

## 七、第六条深线：弱极值原理的"内部极值二阶条件"

纪要里"极值点二阶条件与拉普拉斯算子"是弱极值原理的底层证明。

若 $u$ 在内部点 $x_0$ 取得最大值，则：
$$\nabla u(x_0) = 0, \qquad D^2 u(x_0) \le 0.$$

因此 Hessian 的迹满足 $\Delta u(x_0) = \operatorname{tr} D^2 u(x_0) \le 0$。

对算子 $Lu = -\Delta u + c(x) u$，$c \ge 0$，若 $u(x_0) \ge 0$，则 $Lu(x_0) \ge 0$。这和假设 $Lu < 0$ 矛盾。

**学习体会**：最大值原理不是玄学，而是"二阶导数符号 + 方程符号"的直接结果。考试写法：

1. 反设内部点取非负最大；
2. 写 $\nabla u = 0$，$D^2 u \le 0$，$\Delta u \le 0$；
3. 因 $c \ge 0$、$u \ge 0$，得 $-\Delta u + c u \ge 0$；
4. 与 $Lu < 0$ 矛盾。

---

## 八、第七条深线：一般极值原理靠"扰动辅助函数"消除非严格性

如果假设只是 $Lu \le 0$，不严格，内部最大值处可能只能推出 $0 \le 0$，没有矛盾。老师纪要里因此讲了辅助函数法。

典型套路：$w = u + \varepsilon \phi$，其中 $\phi$ 选成指数函数或线性/二次函数，使得 $Lw < 0$。

这样就把非严格不等式变成严格不等式，回到已经证明的弱极值原理。最后令 $\varepsilon \to 0$，回到 $u$。

**学习体会**：辅助函数的核心作用是把弱不等式变成严格不等式，从而触发弱极值原理；最后让扰动消失。考试如果问"为什么要构造 $w = u + \varepsilon \phi$"，就答这句话。

## 九、第八条深线：Hopf 引理的球壳障碍函数

老师纪要中有一大段：

- 构造球壳；
- 定义辅助函数 $W$；
- 设计径向函数 $V$；
- 分析 $V$ 在内外边界的符号；
- 排除内边界最大；
- 最后求方向导数。

这正是教材证明 Hopf 引理的标准路线。

径向障碍函数取 $V(x) = |x|^\alpha - R^\alpha$ 或指数型径向函数。关键性质：

1. $V = 0$ 在外边界；
2. $V < 0$ 在球壳内部；
3. $LV$ 的符号可由参数控制；
4. $\nabla V$ 在边界方向与法向量一致或相反，法向导数可显式计算。

构造 $W = u - u(x_0) + \varepsilon V$，然后：

- 证明 $W$ 满足极值原理条件；
- 证明最大值不能在内边界；
- 所以最大值在外边界点 $x_0$；
- 对 $W$ 沿适当方向取导数，推出 $\partial_n u(x_0) > 0$。

### 与期中计算题的连接

这就是为什么前面反复练：
$$n(x) = \frac{x}{R}, \qquad \nabla \psi(|x|) = \psi'(|x|) \frac{x}{|x|}, \qquad \partial_n \psi(|x|) = \psi'(R).$$

球面外法向、径向函数法向导数，不只是计算题，而是 **Hopf 引理证明中的技术部件**。

## 十、第九条深线：强极值原理的"开闭集 + 连通性"

老师纪要里强极值原理证明的后半段很关键。

设 $M = \max_\Omega u$，$O = \{x \in \Omega : u(x) = M\}$。

证明思路：

1. $O$ 非空，因为假设内部达到最大值。
2. $O$ 是闭集，因为 $u$ 连续。
3. 要证明 $O$ 是开集——由平均值公式。
4. 若 $O$ 不是开集，则存在最大值点附近能找到非最大值点。
5. 利用球域方向导数引理/Hopf 型结论推出矛盾。
6. 因 $\Omega$ 连通，既开又闭的非空子集只能是整个 $\Omega$。
7. 所以 $u \equiv M$。

老师特别追问"联通是什么意思""开集/闭集原像怎么判断"，说明他很重视证明中的拓扑基础。

**学习体会**：

- 为什么 $O$ 是闭集？答：$O = u^{-1}(\{M\})$，$\{M\}$ 闭，$u$ 连续。
- 为什么连通性必要？答：若区域不连通，函数可在一个连通分支为常数最大，在另一个分支不同，不必全域常数。

## 十一、第十条深线：能量估计是从"点态"走向"积分"的分水岭

老师纪要中对能量估计的动机讲得很清楚：

- 对调和函数，点态工具很强：平均值、最大值原理、导数估计。
- 对更一般的 PDE，特别是非线性方程，点态估计不一定好做。
- 于是转向积分型控制，也就是能量估计。

典型能量：$\int_\Omega u^2 \, dx$，$\int_\Omega |\nabla u|^2 \, dx$。

基本方法：

1. 方程两边乘 $u$ 或测试函数；
2. 在区域上积分；
3. 用 Green 第一公式/积分分部；
4. 得到梯度平方积分；
5. 用 Cauchy、Young、Poincaré/Friedrichs 控制。

### 课堂讲的半径能量单调性

纪要里出现"能量随半径递增"的证明。老师引导：

- 要证明关于 $R$ 递增，就对 $R$ 求导；
- 但 $R$ 出现在积分区域上；
- 所以用变量替换把 $B_R$ 或 $\partial B_R$ 变成单位球/单位球面；
- 再对被积函数求导；
- 最后用调和方程、散度定理、法向导数平方得到非负性。

**这和平均值公式证明共用同一个技巧**——把变动半径变成单位球，让 $R$ 进入被积函数。

## 十二、老师的证明偏好：不要只背结论，要背"触发条件"

从纪要可以看出，老师反复追问的是：

1. 条件是否足够使用定理；
2. 积分区域是否有奇点；
3. 能否交换极限、积分、求导；
4. 边界项为什么消失；
5. 法向量方向有没有写对；
6. 严格/非严格不等式如何处理；
7. 连通性、有界性、光滑边界的作用是什么。

所以答证明题时要主动写：

- $\Omega$ 光滑有界，保证散度定理/Green 公式；
- $u \in C^2(\Omega) \cap C(\overline\Omega)$，保证二阶导和边界最大值；
- $c(x) \ge 0$，保证最大值原理符号；
- $G = 0$ on $\partial\Omega$，保证边界项消失；
- $f$ 紧支撑/光滑，保证卷积和换序合法；
- $x \ne y$ 或挖掉 $B_\varepsilon$，避免基本解奇点。

## 十三、四类 PDE 方法的精神

这批纪要紧扣教材和讲义，不是让你记一堆公式，而是让你掌握四种 PDE 方法：

1. **平均值方法**：调和函数的值由周围平均决定，所以它刚性强。
2. **积分表示方法**：用基本解、Green 函数、Poisson 核把解写出来。
3. **比较/极值方法**：用符号和边界控制解，证明唯一性和稳定性。
4. **能量方法**：当点态控制不够时，用积分控制解和梯度。

期中最可能考的是第 2、3、4 类方法的交叉：  
Green 公式推表示/相容条件，最大值原理推唯一性/估计，能量法推基础先验估计。

## 十四、复习路线建议

### 第一遍：背主线

从"适定性"入手：存在性靠表示公式；唯一性靠最大值原理；稳定性靠最大模估计；正则性靠平均值/导数估计/能量。

### 第二遍：背五个证明模板

1. Neumann 相容条件。
2. Dirichlet 唯一性。
3. Green 函数表示公式。
4. 最大模估计。
5. 能量估计。

### 第三遍：补两个深证明

1. 导数估计：$k=0$、$k=1$、归纳嵌套球。
2. 强极值/Hopf：辅助函数、球壳、开闭集。

### 第四遍：整理一页"条件表"

| 定理 | 必须条件 | 最容易漏 |
|---|---|---|
| Green 公式 | 区域光滑、函数足够光滑 | 外法向方向 |
| 最大值原理 | $c \ge 0$、$u \in C^2 \cap C$、不等式方向 | 非负最大值 |
| 强极值原理 | 区域连通 | 内部取最大才推出常数 |
| Hopf 引理 | 边界严格极值、内球条件 | 外法向导数符号 |
| Green 函数表示 | $G = 0$ on 边界，$-\Delta G = \delta$ | 固定 $x$，对 $y$ 用公式 |
| 能量估计 | 边界项消失或可控 | 测试函数选择 |

## 十五、最浓缩的"老师意图"

老师讲课的深层主线是 $\text{存在性} + \text{唯一性} + \text{稳定性} + \text{正则性}$。每个工具都对应适定性的一部分：

- **Poisson 核 / Green 函数**：构造解，解决存在性；
- **最大值原理**：证明唯一性；
- **最大模估计**：证明稳定性；
- **导数估计 / 能量估计**：分析正则性。

**最深的学习心得**：PDE 不是一堆孤立公式，而是一套"问题—工具"的对应体系。看到任何新方程，先问"它有哪种适定性问题？哪种工具最自然？"这才是老师想让你建立的 PDE 观念。
