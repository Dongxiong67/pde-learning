# 期中复习

> 基于上次期中考题风格 + 板书 + 复习提纲整理。

## 命题风格总结

| 题型 | 内容 | 比重 |
|---|---|---|
| 填空/简答 | 方程阶数与分类、向量恒等式、基本概念 | 约 20% |
| 计算 | 球面法向导数、散度定理应用、基础积分 | 约 30% |
| 证明 | 最大值原理、Green 公式、相容条件、能量估计 | 约 50% |

证明题老师看的是：**条件是否写全**（区域光滑、有界、系数符号条件）、**关键一步是否成立**（最大值原理的触发条件、散度定理的使用条件）。

## 高频考点清单

### 第一类：基础题（填空/简答）

- [ ] PDE 阶数判断（看最高阶偏导数）
- [ ] 线性/半线性/拟线性/非线性 分类
- [ ] 典型方程名称：Laplace / Poisson / Heat / Wave / KdV / Burgers
- [ ] 梯度、散度、旋度、拉普拉斯算子的记号
- [ ] 向量恒等式：$\nabla \times (\nabla \varphi) = 0$，$\nabla \cdot (\nabla \times F) = 0$

### 第二类：计算题

- [ ] 球面外法向的表达式
- [ ] 径向函数的梯度与法向导数
- [ ] 用散度定理把体积分换成边界积分
- [ ] Green 公式的直接应用

### 第三类：证明题（核心）

- [ ] **Neumann 相容条件**：$\int_\Omega f + \int_{\partial\Omega} g = 0$
- [ ] **Dirichlet 唯一性**：作差 + 最大值原理
- [ ] **最大模估计**：障碍函数法
- [ ] **Hopf 引理**：障碍函数 + 反证
- [ ] **Green 函数表示公式**：Green 第二公式 + $\delta$ 函数
- [ ] **能量估计**：分部积分 + Friedrichs / Poincaré

## 各章复习重点

### Ch1 基本概念

- 核心是**工具箱**：散度定理、Green 公式、向量恒等式
- 这些是所有证明题的"基建"，必须熟练

### Ch2 位势方程（期中核心）

三条主线必须打通：

1. **Green 函数体系**：基本解 → Green 函数 → 表示公式 → Poisson 核
2. **最大值原理体系**：弱/强最大值原理 → 比较原理 → Hopf 引理 → 最大模估计
3. **能量法体系**：分部积分 → 测试函数 → 能量不等式

### 模型方程识别

- Porous medium equation：$u_t - \Delta(u^\gamma) = 0$，2 阶，退化抛物
- KdV：$u_t + uu_x + u_{xxx} = 0$，3 阶，非线性色散

## 复习建议（按"先拿分后补全"）

**第 1–2 天**：通读知识梳理，把符号/常用公式统一
- Green 公式、Hopf 引理表述、Poisson 核/Green 函数定义

**第 3–5 天**：按 [解题模板](../03_output/templates) 训练证明题写法
- 每题只练"骨架 + 关键不等式"，不用完整写

**第 6–8 天**：训练计算题与填空
- 向量恒等式、法向导数、基本方程分类

**第 9 天**：做模拟卷（限时），对照答案修正书写与逻辑

**第 10–14 天**：二刷模板 + 整理自己的"最后一页纸"
- 把每个定理的使用条件写清楚
