# PDE 知识地图

## 三大基本方程

```
PDE 三大基本方程
├── 椭圆型 Elliptic
│   └── Laplace / Poisson 方程（位势方程）
│       ├── 调和函数性质（平均值、最大值原理）
│       ├── 基本解 + Green 函数
│       ├── 极值原理 / 比较原理 / Hopf 引理
│       └── 能量估计
│
├── 抛物型 Parabolic
│   └── 热方程 Heat Equation
│       ├── 基本解（Gauss 核）
│       ├── 最大值原理
│       └── 能量估计
│
└── 双曲型 Hyperbolic
    └── 波动方程 Wave Equation
        ├── d'Alembert 公式
        ├── 特征线法
        └── 能量估计
```

## 四大研究问题

$$\text{存在性} \quad+\quad \text{唯一性} \quad+\quad \text{稳定性} \quad+\quad \text{正则性}$$

| 问题 | 工具体系 |
|---|---|
| **存在性** | 基本解 → Green 函数 → 表示公式；变分法 / 能量法 |
| **唯一性** | 最大值原理；能量法（作差估计） |
| **稳定性** | 最大模估计 / 比较原理；能量估计 |
| **正则性** | 平均值性质 → 导数估计；Schauder 估计 |

## 三类证明套路

1. **作差法**：两个解的差满足零边界条件，用最大值原理或能量估计推出差为 0
2. **障碍函数法**：构造辅助函数（如 $|x|^2$），与比较原理配合得到估计
3. **分部积分法**：乘测试函数 → 分部积分 → 用不等式控制

## 核心工具箱

- 散度定理（Gauss）
- Green 第一公式、第二公式
- Friedrichs / Poincaré 不等式
- Cauchy-Schwarz / Young 不等式
